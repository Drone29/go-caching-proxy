# caching-proxy

Caching proxy server

# Usage

```sh
caching-proxy --port <number> --origin <url>
```

# Build And Install

To build and install, use `go build` and `go install` respectively, from the project's root directory 
```sh
go build
```
```sh
go install
```

# Testing

```sh
go test ./...
```
